### 🚀 Quick Start Guide
1.  **Install:** Get the script via **GitHub** or **GreasyFork**. Ensure you have a userscript manager (like Tampermonkey or Violentmonkey) installed.
2.  **Navigate:** Open any gallery on E-Hentai.
3.  **Download:** Locate the "E-Hentai Downloader" box and click **"Download Archive."**
4.  **Wait & Save:** Once the script finishes processing, your browser will prompt you to save the zip file.

**Pro-Tips:**
* **Number Images:** Check this option to keep images in order.
* **Pages Range:** Use this to download specific parts of a gallery (highly recommended for large sets).
* **Settings:** Visit this tab for deeper customization.

---

### 🛠 How It Works
Unlike the official download button, this script **does not cost GP or Credits**. 
1. It fetches every image URL in the gallery.
2. It downloads the images individually via your browser.
3. It packages them into a single zip file using **JSZip** and **FileSaver.js**.

### 💻 System Requirements
The script's compatibility depends on your browser and extension manager.

| Browser | Supported Managers | Notes |
| :--- | :--- | :--- |
| **Chrome** | Tampermonkey (5.0+), Violentmonkey | Enable **Developer Mode** for Manifest v3. |
| **Firefox** | Greasemonkey (4.1+), Tampermonkey, Violentmonkey | Best performance on desktop. |
| **Edge** | Tampermonkey, Violentmonkey | Update Windows 10 to at least build 14393. |
| **Safari** | Tampermonkey (4.3+) | Requires macOS 10.12.4+. |
| **Mobile** | Kiwi, Yandex, Firefox Nightly | Functional, but limited by mobile RAM. |
